# Git Task Plugin

A command plugin for Git (that's it).

---

## Requirements

- A unix-based OS

## Installation

- `git clone https://github.com/SamuelDenani/git-task-plugin`
- `cd git-task-plugin`
- `sudo sh ./install.sh`

## Usage

`git task [-i] <type> <issue-ID> <description>`

> Example: git task feat JIRA-123 description -i
